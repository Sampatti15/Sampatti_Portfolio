# 🌐 Sampatti Portfolio

Welcome to my personal portfolio website! This project showcases my skills, projects, and journey as an aspiring Machine Learning Engineer and Frontend Developer.

## 🚀 Live Demo

🔗 https://sampatti-portfolio.netlify.app/#

## 📌 About

This portfolio serves as a central hub to present my work, technical skills, and achievements. It highlights the projects I've built, the technologies I'm learning, and provides an easy way for recruiters and collaborators to connect with me.

## ✨ Features

- Responsive design for desktop and mobile
- Clean and modern user interface
- About Me section
- Skills showcase
- Featured Projects
- Contact information
- Smooth scrolling navigation
- Professional layout and animations

## 💻 Getting Started

Clone the repository

```bash
git clone https://github.com/Sampatti15/Sampatti_Portfolio.git
```

Navigate to the project folder

```bash
cd Sampatti_Portfolio
```

Open the project

Simply open `index.html` in your browser.
